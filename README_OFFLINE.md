### 第1步：准备离线包
```
将以下文件夹复制到目标电脑：

```

### 第2步：启动应用
```
双击运行：offline_start.cmd
```

### 第3步：等待初始化（仅首次）
```
✓ 正在创建虚拟环境...
✓ 正在离线安装依赖...

```
首次初始化约2-3分钟，之后秒开。

```
dist_offline_complete/

└── .venv/                     # 自动创建
```

## 🔧 系统要求

### 最低要求：
- **操作系统**: Windows 10+
- **内存**: 4GB RAM
- **存储**: 3GB 可用空间
- **Python**: 已安装Python 3.11+（推荐）

### 理想配置：
- **操作系统**: Windows 11
- **内存**: 8GB RAM
- **显卡**: NVIDIA GPU（支持CUDA加速）
- **存储**: SSD硬盘

## 🎮 使用流程

1. **启动软件**：双击 offline_start.cmd
2. **选择音频**：支持MP3/WAV/M4A等格式
3. **选择模型**：根据需要选择基础或大型模型
4. **开始转写**：点击开始，实时显示转写结果
5. **导出结果**：支持导出为TXT文本文件

## 🚨 注意事项

- **首次启动**会自动创建虚拟环境并安装依赖
- **模型文件**较大，请确保磁盘空间充足
- **GPU加速**：如系统支持会自动启用CUDA
- **静音环境**：建议在安静环境中使用以获得最佳效果
